Scene
Sceneはシーングラフを構築するトップレベルコンテナです. Sceneは複数のLayerを管理します. このクラスを使用して、画面を構築します. デフォルトのレイヤー（topLayer）を持ちます. DisplayObjectの親に直接指定された場合、実際に追加される先はtopLayerになります. Sceneのライフサイクルについて 1. onCreate() ... 生成時に呼ばれます. 2. onStart() ... 開始時に呼ばれます. 3. onResume() ... 再開時に呼ばれます. 4. onPause() ... 一時停止時に呼ばれます. 5. onStop() ... 終了時に呼ばれます. 6. onDestroy() ... 破棄時に呼ばれます.
