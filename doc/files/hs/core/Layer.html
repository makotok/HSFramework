Tables

Layer
シーンに追加できるレイヤークラスです レイヤーは、ビューポートを持ち、MOAILayerに対応するクラスとなります. レイヤー内の子オブジェクトの移動は、カメラを使用して移動してください. レイヤーの直接的な移動は、画面内の座標を移動する事になります.
